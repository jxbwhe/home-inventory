鉴权说明： 所有 /openapi/* 接口均需要在 HTTP Header 中携带 Token。
格式：X-API-Token: 你的Token
POST /openapi/usages

记录物品使用消耗。

请求 Body (JSON):
{ "item_id": 1, "quantity": 1.5 }
POST /openapi/purchases

记录物品购买。

请求 Body (JSON):
{ "item_id": 1, "quantity": 10, "price": 25.5 }
POST /openapi/items

创建新物品。

请求 Body (JSON):
{ "name": "抽纸", "description": "清风抽纸 3层", "unit": "包", "min_quantity": 2, "family_id": 1 /* 可选 */ }
GET /openapi/items

获取所有物品列表。

请求参数查 (Query):
  • family_id (可选): 过滤指定家庭下的物品
GET /openapi/families

获取家庭列表（适合快捷指令第一层菜单）。

GET /openapi/shortcut/menu

获取快捷录入菜单数据，包含动作、家庭和物品（支持 family_id 过滤物品）。

返回示例:
{ "actions": [{"value":"purchase","label":"补货"},{"value":"usage","label":"使用"}], "families": [{"id":1,"name":"家里","item_count":6}], "items": [{"id":3,"name":"抽纸","family_id":1,"unit":"包","current_quantity":2}] }
POST /openapi/quick-entry

快捷统一录入接口（快捷指令建议优先调用它）。

请求 Body (JSON):
{ "action": "purchase", // purchase 或 usage "family_id": 1, // 可选 "item_id": 3, // 推荐使用 "quantity": 4, "price": 22.5, // action=purchase 必填 "note": "快捷指令录入" }
iPhone 快捷指令推荐流程
  1. 调用 GET /openapi/families，让用户选择家庭。
  2. 调用 GET /openapi/items?family_id=选中家庭，让用户选择物品。
  3. 菜单选择动作（补货/使用），并让用户输入数量；补货时再输入金额。
  4. 调用 POST /openapi/quick-entry 提交记录。
  5. 展示返回结果中的 current_quantity 作为成功提示。
