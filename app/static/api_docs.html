鉴权说明： 所有 /openapi/* 接口均需要在 HTTP Header 中携带 Token。
格式：X-API-Token: 你的Token
POST /openapi/usages

记录物品使用消耗。

请求 Body (JSON):
{ "item_id": 1, "quantity": 1.5 }
POST /openapi/purchases

记录物品购买。

请求 Body (JSON):
{ "item_id": 1, "quantity": 10, "price": 25.5 }
POST /openapi/items

创建新物品。

请求 Body (JSON):
{ "name": "抽纸", "description": "清风抽纸 3层", "unit": "包", "min_quantity": 2, "family_id": 1 /* 可选 */ }
GET /openapi/items

获取所有物品列表。

请求参数查 (Query):
  • family_id (可选): 过滤指定家庭下的物品
